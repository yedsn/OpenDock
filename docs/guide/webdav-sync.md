# WebDAV 同步

OpenDock 内置 WebDAV Sync 插件，支持将工作空间数据同步到用户自有的 WebDAV 端点。

## 概览

- 同步方式：双向同步
- 存储后端：WebDAV
- 数据格式：JSON
- 插件位置：`plugins/.system/webdav-sync/`

## 使用前提

调用前需要满足：

- OpenDock 正在运行
- 已安装并启用 WebDAV Sync 插件
- 有一个可用的 WebDAV 端点（地址、用户名、密码）

## 配置 WebDAV

1. 打开设置页
2. 找到 WebDAV Sync 插件设置
3. 填写 WebDAV 地址、用户名和密码
4. 点击"测试连接"确认可用
5. 启用同步

## 同步范围

WebDAV 同步会处理以下数据：

- 工作空间配置
- 场景定义
- 集合和集合项
- 工具配置
- 插件数据（不含凭据明文）

## 凭据安全

导出数据中不包含插件凭据明文。配置对象中只保留 `credentialRef` 引用，真实值保存在用户数据目录的插件数据命名空间里。

## 常见问题

### 连接失败

优先检查：

- WebDAV 地址是否正确
- 用户名和密码是否有效
- 网络是否可达

### 同步冲突

如果本地和远端同时修改了相同数据，需要手动选择覆盖方向。
